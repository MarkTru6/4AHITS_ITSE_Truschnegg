## Übung (netcat chat)

listener:
```
nc -l -p <Port-Nummer>
```
```
nc <IP-Addresse listener> <Port-Nummer listener>
```
![Textchat](img/netcat_textchat.png)

## Übung (netcat data send)

empfänger:
```
nc -l -p <Port-Nummer> > <gewünschter datei name>
```
Sender:
```
nc <IP-Addresse empfänger> <Port-Nummer empfänger> < <Datei Name>
```
